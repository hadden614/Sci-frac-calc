# Mobile Scientific Calculator (iPhone Optimized)

## What it does
• Scientific calculator  
• Trigonometry (DEG/RAD)  
• Logarithms (log10, ln)  
• Constants π and e  
• Memory-style workflow with history (20 entries)  
• Fraction ↔ decimal conversion  
• Tradesman rounding (nearest 1/16 etc.)  
• Keyboard input supported  

---

## Run locally
1. Download repo
2. Open folder
3. Double-click `index.html`

No install required.

---

## Deploy with GitHub Pages
1. Create a GitHub repo
2. Upload all files
3. Go to **Settings → Pages**
4. Source: `Deploy from branch`
5. Branch: `main`
6. Folder: `/root`
7. Save

Your calculator will appear at:

https://YOUR-USERNAME.github.io/REPO-NAME

---

## Keyboard shortcuts
Enter → equals  
Backspace → delete last  
Numbers & operators → type normally

---

## Known limitations
• Parser is simple (no matrices)
• No persistent memory registers
• Very complex nested expressions may lose precision
